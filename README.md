# 🎯 媒体嗅探器 Pro (Media Sniffer Pro)

> 一个功能强大的浏览器油猴脚本，自动嗅探页面中的图片、视频、音频、流媒体等媒体资源，支持一键下载和批量导出。

## ✨ 功能特性

### 🖼 图片嗅探
- 自动扫描页面所有 `<img>` 标签
- 扫描 CSS `background-image` 背景图片
- 懒加载图片自动检测
- 按尺寸/格式筛选
- 批量下载

### 🎬 视频嗅探
- 自动检测页面视频元素
- 支持 MP4 / WebM / MOV 等格式
- **视频封面自动提取显示**
- **视频页面链接智能解析**（B站/抖音/快手等）
- 多清晰度切换（8K/4K/1080P/720P）

### 🔗 视频页面链接解析

| 🎵 抖音 (douyin) | ✅ 完整支持 | 无水印视频提取 |
| ⚡ 快手 (kuaishou) | ✅ 完整支持 | 无水印视频提取 |
| 📕 小红书 | 🔧 开发中 | 链接识别已完成 |

| 💬 微信视频号 | 🔧 开发中 | 链接识别已完成 |

### 📱 移动端优化
- 响应式设计，手机端完美适配
- **长按卡片弹出操作菜单**（预览/复制/下载/打开）
- **预览弹窗下滑关闭**

## 🚀 安装方法

### 前置要求
- Chrome / Edge / Firefox 浏览器
- 已安装 **Tampermonkey** 或 **ScriptCat** 扩展

### 一键安装
1. 点击脚本文件：[`media-sniffer-v8.2.0bate.user.js`](media-sniffer-v8.2.0bate.user.js)
2. Tampermonkey / ScriptCat 会自动检测并提示安装
3. 点击「安装」即可

### 手动安装
1. 复制脚本全部代码
2. 打开 Tampermonkey → 「添加新脚本」
3. 全选删除原有代码，粘贴新代码
4. `Ctrl + S` 保存

## 📖 使用说明

### 基本操作
- **点击紫色浮动按钮** → 打开/关闭侧边栏
- **拖动紫色按钮** → 拖动到任意位置（自动记忆）
- **普通点击资源** → 预览
- **双击资源** → 直接下载
- **Shift + 点击** → 多选批量下载
- **快捷键** `Alt + B` → 快速打开/关闭
- **按 Esc** → 关闭面板

### 视频链接解析
1. 打开视频网站首页/列表页
2. 点击右下角紫色按钮
3. 切换到「视频」标签
4. 顶部显示「视频页面链接」区域
5. **点击卡片** → 解析并预览
6. **一键解析全部** → 批量解析所有视频
